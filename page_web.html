Introduction

Depuis le lycée, et depuis tout petit, je suis un grand fan de jeux-vidéos et de la diversité de pages web qu'on retrouve sur le net. J'aime l'accessibilité qu'offre la création de page web et la beauté de ce qui en ressort. C'est pourquoi dans cette page web, je présenterai le métier de développeur web, métier assez commun selon moi mais qui demande une certaine rigueur dans son travail. Nous allons donc explorer les problématiques suivantes :

- Quels sont les missions et les responsabilités d'un développeur web ?

- Quel niveau d'études sont recommandés ?

Conclusion

Comme nous avons pu le voir, le métier de développeur web est dynamique et demande une constante vigilance, aussi bien pour le cahier des charges que pour les diverses avancés technologiques. En se basant sur les différents types de ressources que nous vous avons présentés, il faut se spécialiser sur plusieurs langages tel que le Javascript, HTML, ou encore Python pour ne citer que les plus connus. Ce choix de métier est bien choisi puisque les demandes se font de plus en plus et le salaire suit avec, car en moyenne un débutant aura un contrat de 29 000€/an.
